This sample webpage uses a google map api. i have decided the coordinates below. Enjoy

the API in question: https://maps.googleapis.com/maps/api/js?key=YOUR_API_KEY&callback=initMap

    This map works through a few builtin functions and variables. initMAP, map, options, and marker
  • the "initMAP" function is the callback function searched for by the API
  • the variable "map" searches the document for an id of "map" and takes an argument of "options"
  • the variable "options" allows me to specify where and how i like my map
  • the variable "marker" allows me to set a point of interest in my map, such as an address
